Supabase接続設定

Supabaseに接続することで、データがクラウドに保存され、スマホや他の家族の端末とリアルタイムで同期できます。

接続手順:
  1. Supabaseの管理画面にログインします。
  2. Project Settings > API を開きます。
  3. URL と anon public キーをコピーしてここに貼り付けます。

AI (Gemini) 設定

Google AI Studioから取得したGemini APIキーを設定してください。キーはローカル（localStorage）にのみ保存されます。

  1. Google AI Studio にアクセスします。
  2. 「Get API Key」をクリックして、キーを作成します。
  3. 作成したキーをコピーしてここに貼り付けます。
